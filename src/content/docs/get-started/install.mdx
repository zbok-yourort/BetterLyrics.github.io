这一栏目将会教你怎么安装BetterLyrics。

# 通过Microsoft Store安装（推荐）
<a href="https://apps.microsoft.com/detail/9P1WCD1P597R?referrer=appbadge&mode=direct"><img src="https://get.microsoft.com/images/en-us%20dark.svg" width="200"/></a>

通过Microsoft Stroe安装可及时享受到BetterLyrics的功能更新和Bug修复。
商店免费试用版和付费版**并无本质差异**，购买付费版视为对开发者的支持。

# 从Microsoft Store下载BetterLyrics的安装包并安装

点击[此处](https://store.rg-adguard.net/)打开第三方Microsoft Store的软件包提取器，将BetterLyrics的链接`https://apps.microsoft.com/detail/9P1WCD1P597R`粘贴进输入框中

![截图](https://free.boltp.com/2026/02/04/69822778c92d8.webp)

选择`XXXXX.BetterLyrics_X.X.XXX.X_neutral_~_XXXXXXXXXXX.msixbundle`这样的文件（图片仅作参考，请以实际为准）
下载后只需双击打开安装即可，若电脑带有Micorsoft Store也可及时享受到开发者的功能更新和Bug修复。

# 使用侧载安装BetterLyrics
如果前两种方案均无法正常安装或者您希望尝试新版本，可按如下操作：

1.在[此处](https://github.com/jayfunc/BetterLyrics/releases/latest)下载BetterLyrics的最新版本。请下载以`.cer`后缀的证书文件和`.msixbundle`后缀的软件包（如图所示）：
![Release界面示例](https://free.boltp.com/2026/02/04/69834ef808cf2.webp)

如果您下载的是ZIP压缩文件，证书文件和软件包已包含其中，**直接下载并解压**即可。

2.双击证书文件，在弹出的界面中点击**安装证书**。
![证书](https://free.boltp.com/2026/02/04/6983503454268.webp

3.在**证书导入向导**窗口中，存储位置选择**“当前用户”**，并点击下一步。
![证书导入向导_当前用户](https://free.boltp.com/2026/02/04/6983519924f21.webp)

4.选择**将所有证书都放入下列存储**
![证书存储](https://free.boltp.com/2026/02/04/698351992ac62.webp)

点击**浏览**按钮后，在弹出的窗口中选择**受信任的根证书颁发机构**，点击确定。
![选择证书存储](https://free.boltp.com/2026/02/04/6983523d37768.webp)

5.一直下一步，直至弹出**安全警告**，点击**是**。

6.弹出`导入成功`后，双击软件包便可直接安装

⚠️该方法无法自动更新BetterLyrics且**存在一定风险**，若非必要请尽可能选择前两种方式。
