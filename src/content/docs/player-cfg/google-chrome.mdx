> [Google Chrome](https://www.google.com/chrome) is a cross-platform web browser developed by Google. It was launched in 2008 for Microsoft Windows, built with free software components from Apple WebKit and Mozilla Firefox. Versions were later released for Linux, macOS, iOS, iPadOS, and Android, where it is currently the default browser. The browser is also the main component of ChromeOS, on which it serves as the platform for web applications.[^1]
