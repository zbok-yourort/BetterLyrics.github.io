> 数十万用户选择的本地音乐播放器品牌[^1]

## Setup Guide

**No configuration required**. This player natively supports System Media Transport Controls (SMTC). Simply start playback to use.

### Known Limitation

**Seek Control** is not supported (Progress is read-only; dragging to seek is unavailable).
