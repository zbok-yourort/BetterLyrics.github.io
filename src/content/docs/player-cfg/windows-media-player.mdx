> [Windows Media Player](https://apps.microsoft.com/detail/9wzdncrfj3pt) (WMP, now referred to as Windows Media Player Legacy officially to distinguish it from its newer counterpart introduced in Windows 11) is the first media player and media library application that Microsoft developed to play audio and video on personal computers. It has been a component of the Microsoft Windows operating system, including Windows 9x, Windows NT, Pocket PC, and Windows Mobile. Microsoft also released editions of Windows Media Player for classic Mac OS, Mac OS X, and Solaris, but has since discontinued them.[^1]

## Setup Guide

Only supports the new **Media Player**. Does **NOT** support the legacy **Windows Media Player**.
