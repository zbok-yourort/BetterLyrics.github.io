# Welcome to BetterLyrics

** 一款优雅且高度自定义的歌词可视化与全能音乐播放应用 **

**本欢迎页旨在帮助用户快速定位且解决遇到的问题：**

BetterLyrics 的运行完全依赖于 Windows 的 **系统媒体传输控制 (SMTC)** 协议。

## 🛠️ 问题简单排查

如果打开 BetterLyrics 后并没有显示歌词，或歌词文本或进度错误，请按照以下思路进行自查：

### 1️⃣ 第一步：检查播放源
**您的播放器是否向 Windows SMTC 汇报了状态？**

* **桌面客户端 (QQ音乐 / 网易云)**：
    * 很多播放器需要额外配置，务必确保按照说明安装对应的插件，或者修改播放器对应设置。
    

### 2️⃣ 第二步：检查软件设置
**BetterLyrics 是否忽略了该应用？**

* 进入 BetterLyrics **设置** -> **播放源**。
* 检查您的播放器名称是否在播放源列表内，并且启动对新播放源的监听选项为打开状态。

## 🚀 快速上手建议

为了获得 **“开箱即用”** 的丝滑体验，我们推荐搭配以下环境：

| 平台 | 推荐客户端 | 说明 |
| :--- | :--- | :--- |
| **流媒体** | **QQ音乐**  | 官方客户端对 SMTC 支持最好，配合插件即可精准匹配。 |
| **流媒体** | **AppleMusicplayer**  | 官方客户端对 SMTC 支持良好，无需额外配置。 |
| **本地音乐** | **BetterLyrics 内置播放器** | 直接拖拽文件播放，支持 LRC/ESLRC 完美解析。 |
