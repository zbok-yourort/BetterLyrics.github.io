title: Windows Media Player
---

> [Windows Media Player](https://apps.microsoft.com/detail/9wzdncrfj3pt)（Windows Media Player）是微软公司出品的一款免费媒体播放器，通常简称“WMP”，又称“WMPlayer”。内置Windows操作系统中，也可以透过网络下载，支持通过插件增强软件功能。从Windows Media Player 7及以后的版本支持更换界面外观。此软件最新版本是于2009年发布的Windows Media Player 12。自2022年起，它于Windows 11中被打上“旧版”前缀（“Legacy”后缀），以区别新的Windows媒体播放器。[^1]

## 适配说明

仅支持 **新版媒体播放器** (Media Player)。**不支持** 旧版 Windows Media Player (Legacy)。

[^1]: [Wikipedia](https://zh.wikipedia.org/wiki/Windows_Media_Player)
